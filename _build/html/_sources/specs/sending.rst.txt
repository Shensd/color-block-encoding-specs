--------

Calibration
-----------

The calibration period is executed before sending data, and it establishes the
following information with the receiving client(s):

* Transmission Speed
* Shift Values

  * Raw color shift
  * Individual numeric values
* Number of unique blocks

Information that is assumed to already be known by the client that is not 
communicated is as such:

* Video source size
